# **App Name**: ClarityScan

## Core Features:

- Automated Lesion Detection: Upload a JPG or PNG image, then leverage AI to automatically identify and segment potential lesions indicative of conditions like macular edema or hemorrhage, using a U-Net based model.
- Lesion Measurement: Calculate measurements (height, width) of detected lesions. The AI model functions as a tool to provide quantitative analysis that supports diagnosis.
- Annotation Overlay: Visually represent detected lesions on the uploaded image with colored overlays and labels (e.g., 'Diffuse Edema').
- Results Export: Enable exporting of the annotated image and measurements in a structured PDF report.
- Image Upload: Provide a simple drag-and-drop interface for users to upload retinal images.

## Style Guidelines:

- Primary color: A calming yet assertive blue (#4285F4), evoking trust and clarity, appropriate for diagnostic tools.
- Background color: A light, desaturated blue (#E3F2FD), maintaining focus on the image data.
- Accent color: An energetic orange (#FF5722), highlighting important calls to action and warnings.
- Body text: 'Inter', a grotesque-style sans-serif providing a modern, machined, objective, neutral, clear reading experience
- Headline text: 'Space Grotesk', for short bursts of impactful information.
- Employ a set of easily recognizable icons, styled with the primary color, to indicate different lesion types.
- Maintain a clean, functional, and spacious layout that minimizes clutter and maximizes screen real estate for image viewing.
